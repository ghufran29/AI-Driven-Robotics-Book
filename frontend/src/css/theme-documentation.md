# Cyber-Physical AI Theme Documentation

## Overview
The Cyber-Physical AI theme implements a modern dark-first design with deep slate (#2a2f35) backgrounds and electric blue (#00eeff) accents, optimized for technical documentation in robotics and AI fields.

## Color Palette
- **Primary**: #00eeff (Electric Blue) - Used for links, highlights, and interactive elements
- **Secondary**: #39ff14 (Neon Green) - Used for success states and positive indicators
- **Background**: #2a2f35 (Deep Slate) - Main background color for dark mode
- **Text**: #e6e6e6 (Light Gray) - Primary text color in dark mode

## Typography
- **Body Text**: Inter font family for all text content
- **Code/Monospace**: JetBrains Mono for code blocks and technical content
- **Font Loading**: Optimized with preconnect and display:swap for performance

## Component Styling
### Navigation
- Glassmorphism navbar with backdrop-filter blur effect
- Electric blue accent hover states
- Responsive design with mobile-first approach

### Sidebar
- Custom scrollbar with electric blue thumb
- Active state indicators with left border accent
- Smooth hover transitions

### Code Blocks
- Custom Prism.js theme with cyber-physical color scheme
- Electric blue for operators and variables
- Neon green for strings and built-in functions
- Dark background (#1e1e1e) for contrast

### Admonitions
- Custom styling for info, warning, danger, and success blocks
- High contrast colors that work in both light and dark modes
- Industrial-style appearance for technical documentation

## Responsive Design
- Mobile-first approach with breakpoints at 768px and 996px
- SVG icons scale properly across all screen sizes
- Three-column feature layout collapses to single column on mobile

## Accessibility Features
- Keyboard navigation support with visible focus states
- WCAG AA compliant contrast ratios
- Reduced motion support for users with vestibular disorders
- High contrast mode compatibility
- Screen reader friendly semantic HTML structure
